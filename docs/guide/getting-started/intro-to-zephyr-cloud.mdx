# Introduction to Zephyr Cloud

Zephyr Cloud enhances and accelerates the software development lifecycle (SDLC).

## Core Features

* Enhance your CI/CD pipeline efficiency
* Boost local development speed
* Improve code quality
* Gain insights into distributed applications

## Public Alpha

Zephyr Cloud is currently in public alpha stage. As you evaluate Zephyr Cloud please do not hesitate to reach out to us on [Discord](https://discord.gg/EqFbSSt8Hx) and provide your feedback.
